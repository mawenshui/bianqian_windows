## 一、安装与启动

### MSI 安装包安装

1. 下载 `StickyNote-1.5.5-win64.msi`
2. 双击运行安装向导（中文界面）
3. 选择安装目录，点击"安装"
4. 安装完成后可选创建桌面快捷方式
5. 启动应用，便签自动出现在桌面

### 便携版使用

1. 下载 `StickyNote-v1.5.5-portable.zip`
2. 解压到任意目录（如 `D:\StickyNote`）
3. 运行 `StickyNote.exe` 即可
4. 数据文件保存在 exe 同目录，方便携带

### 首次启动

- 应用启动后自动创建一个默认便签
- 系统托盘（右下角）出现便签图标
- 支持开机自启（在托盘菜单中勾选）

## 二、基本操作

### 便签窗口结构

```
┌──────────────────────────┐
│  [标题输入区]             │  ← 单行编辑，最多50字符
├──────────────────────────┤
│  [格式工具栏]             │  ← A- A+ B I 颜色 等
│  [透明度/置顶/格式化]      │  ← 滑块、复选框
├──────────────────────────┤
│                          │
│    [内容编辑区]           │  ← 富文本，支持 HTML
│                          │
├──────────────────────────┤
│  [标签芯片] [⏰] [🏷]     │  ← 底部工具栏
│  [删除] [?] [隐藏]        │
└──────────────────────────┘
```

### 移动与调整大小

| 操作 | 方法 |
|------|------|
| **移动便签** | 按住空白区域拖拽 |
| **调整大小** | 鼠标移到窗口边缘/四角，出现双向箭头后拖拽 |
| **最小尺寸** | 100 × 120 像素 |

### 隐藏与删除

| 操作 | 方法 |
|------|------|
| **隐藏便签** | 点击底部「隐藏」按钮 / 右键菜单 / 点击窗口 × 按钮 |
| **恢复便签** | 托盘菜单 → 便签 → 点击对应便签 |
| **删除便签** | 点击底部「删除」按钮 / 右键菜单「删除便签」（需确认） |

> 点击 × 按钮不会关闭便签，而是隐藏到托盘（托盘会提示"便签已隐藏"）。

## 三、文字格式与编辑

### 格式工具栏

选中文字后，使用工具栏按钮设置格式：

| 按钮 | 功能 | 说明 |
|------|------|------|
| **A-** | 减小字号 | 最小 6pt |
| **A+** | 增大字号 | — |
| **B** | 加粗 | 切换式开关 |
| **I** | 斜体 | 切换式开关 |
| **A（彩色）** | 字体颜色 | 打开颜色选择器 |

### 智能格式化

开启底部「智能格式化」复选框后，粘贴内容时自动识别并格式化：

- **支持格式**: JSON、HTML、Markdown、XML、CSS、JavaScript、Python、SQL
- **效果**: 自动缩进、添加格式标签（如 `[JSON格式]`）

### 撤销与重做

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 撤销（最多 50 步） |
| `Ctrl+Y` | 重做 |

标题和内容的撤销栈共享，支持恢复光标位置。

## 六、便签管理

### 透明度调节

便签底部工具栏的透明度滑块：

- **范围**: 20% ~ 100%
- **实时生效**，不影响内容编辑

也可通过右键菜单快速设置：透明度 → 100%/90%/80%/70%/60%/50%/40%/30%

### 总在最前（置顶）

- **方法 1**: 勾选底部工具栏「总在最前」复选框
- **方法 2**: 右键菜单 → 总在最前

开启后便签始终显示在其他窗口之上。

### 数据自动保存

- 编辑后 **500ms** 自动异步保存（后台线程，不卡顿）
- 退出应用时同步保存所有便签
- 数据以 JSON 文件保存在 `notes/` 目录

## 七、标签系统

### 为便签添加标签

1. 点击便签底部 🏷 按钮 / 右键菜单「设置标签」
2. 在标签选择器中勾选已有标签
3. 可点击「新建标签」快速创建
4. 一个便签可关联多个标签

### 标签芯片显示

选中的标签以彩色圆角芯片形式显示在便签底部，点击芯片上的 × 可移除。

### 标签管理

托盘菜单 → 管理标签：

| 操作 | 说明 |
|------|------|
| 添加标签 | 输入名称，系统自动分配颜色 |
| 重命名 | 选中后点击「重命名」 |
| 修改颜色 | 双击颜色块 / 点击「改色」 |
| 删除标签 | 从所有便签中移除该标签 |

系统内置 12 种预设颜色，自动循环分配。

### 标签分组查看

托盘菜单 → 标签分组 → 按标签分类查看便签，点击可快速打开。

## 八、定时提醒

### 设置提醒

1. 点击便签底部 ⏰ 按钮 / 右键菜单「设置提醒」
2. 勾选「启用定时提醒」
3. 设置提醒日期和时间
4. 选择重复模式：**一次性** / **每天** / **每周** / **每月**

### 提醒通知

- 到期时系统托盘弹出通知（显示 5 秒）
- 通知消息自动使用便签标题
- 每 30 秒自动检查一次所有便签的提醒状态
- 同一天内不重复触发

### 状态指示

已启用提醒时：
- ⏰ 按钮高亮为蓝色
- 鼠标悬停显示提醒时间

## 十三、窗口定位与贴边隐藏

### 智能定位

新建便签时自动寻找最佳位置（网格布局，从左上角按列优先排列），避免便签重叠。

### 窗口吸附

| 吸附类型 | 触发距离 | 效果 |
|----------|----------|------|
| 屏幕边缘吸附 | ≤ 15 像素 | 自动对齐到屏幕边缘 |
| 便签间吸附 | ≤ 15 像素 | 自动对齐/贴合其他便签边缘 |

### 贴边自动隐藏

将便签拖拽到屏幕边缘（≤ 3 像素）松手后：

1. 便签滑出屏幕（250ms 动画），留下标题标签页
2. **悬停标签页 300ms** → 临时展开预览（离开自动缩回）
3. **点击标签页** → 保持展开（再次点击或点击其他区域缩回）
4. 支持上下左右四个边缘

### 窗口动画

| 场景 | 动画 |
|------|------|
| 显示便签 | 200ms 淡入 |
| 隐藏便签 | 150ms 淡出 |
| 贴边隐藏 | 250ms 滑出屏幕 |
| 恢复便签 | 250ms 滑回原位 |

## 十五、自动更新

### 自动检查

启动后 3 秒后台自动检查 GitHub 新版本（可在设置中关闭）。

### 手动检查

- 托盘菜单 → 检查更新
- 设置 → 更新设置 → 立即检查更新

### 更新流程

1. 检测到新版本 → 弹出对话框显示更新日志
2. 三个选项：**立即更新** / **稍后提醒** / **跳过此版本**
3. 下载进度条显示（支持 GitHub 主链接 + ghproxy 镜像双链路）
4. 下载完成 → 自动退出 → .bat 脚本解压替换 → 重启应用
5. 用户数据（便签、设置、模板）自动保留，不覆盖

### 安装类型自动识别

| 类型 | 更新方式 |
|------|----------|
| MSI 安装 | 下载 MSI 安装包 |
| 便携版 | 下载 ZIP 便携包 |
| 源码运行 | 下载 ZIP 源码包 |

## 十七、常见问题

**Q: 便签数据保存在哪里？**
A: MSI 安装版保存在安装目录的 `notes/` 子目录；便携版保存在 exe 同目录。每个便签为独立 JSON 文件。

**Q: 如何恢复误删的便签？**
A: 如果有备份，可在备份管理中恢复。建议开启自动备份功能。

**Q: 全局快捷键不生效？**
A: 确保应用正在运行（系统托盘可见图标）。如果其他软件占用了相同快捷键，可能会冲突。

**Q: 如何自定义主题？**
A: 在 `styles/` 目录中创建 `.css` 文件，参照现有主题文件的格式编写。保存后重启应用即可在主题列表中出现。

**Q: 更新后便签数据会丢失吗？**
A: 不会。自动更新流程会保留 `notes/`、`settings.json`、`templates/` 等用户数据目录。

**Q: 如何批量导出便签？**
A: 托盘菜单 → 导入导出 → 导出标签页 → 全选 → 选择格式 → 导出。多个便签会自动打包为 ZIP。
