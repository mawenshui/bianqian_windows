## 简介

**StickyNote** 是一款功能丰富的桌面便签应用，主要特性包括：

- 📝 **多便签管理** — 创建、编辑、隐藏、删除便签
- 🎨 **10种主题** — 软黄、经典白、优雅绿、清新蓝、午夜黑、海洋青、暖阳橙、日落橙、活力紫、暖粉
- 🔍 **全文搜索** — 按标题和内容搜索所有便签
- ⏰ **定时提醒** — 一次性/每天/每周/每月提醒，系统托盘通知
- 🏷️ **标签分组** — 12种颜色标签，便签分类管理
- 📋 **便签模板** — 5个内置模板 + 自定义模板
- 📥 **导入导出** — TXT / Markdown / ZIP 格式
- ↩️ **撤销/重做** — 支持 Ctrl+Z / Ctrl+Y
- 💾 **数据备份** — 自动+手动备份，ZIP压缩存储
- 🖱️ **右键菜单** — 快速访问主题、字体、透明度、标签、提醒等设置
- 📌 **窗口吸附** — 屏幕边缘吸附 + 便签间对齐
- ✨ **淡入淡出动画** — 窗口显示/隐藏平滑过渡
- ⚡ **异步I/O** — 后台线程保存/加载，主界面不卡顿

## 系统要求

- **操作系统**：Windows 10 或更高版本
- **运行环境**：Python 3.7+，PyQt5 5.15+
- **存储空间**：至少 100 MB 可用空间

## 核心功能

### 创建便签

- **托盘菜单**：右键托盘图标 → "添加便签"
- **全局快捷键**：`Ctrl+Shift+N`
- **从模板创建**：托盘菜单 → "从模板创建..."

### 编辑便签内容

- 点击标题栏编辑标题，输入完成后自动保存
- 在内容区输入文本，支持多行编辑
- 更改实时保存（500ms 防抖延迟），无需手动保存

### 右键上下文菜单

在便签上右键点击可快速访问：

- **复制** — 复制全部内容到剪贴板
- **粘贴** — 从剪贴板粘贴内容
- **切换主题** — 子菜单列出所有10种主题，当前主题已勾选
- **字体设置** — 增大字体 / 减小字体
- **总在最前** — 切换便签置顶状态
- **透明度** — 互斥选择30%/40%/50%/60%/70%/80%/90%/100%
- **设置标签** — 打开标签选择对话框
- **设置提醒** — 打开提醒设置对话框
- **隐藏便签** / **删除便签**

### 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+N` | 创建新便签 |
| `Ctrl+Shift+F` | 打开搜索对话框 |
| `Ctrl+Shift+B` | 打开备份管理对话框 |
| `Ctrl+Z` | 撤销操作 |
| `Ctrl+Y` | 重做操作 |

### 调整便签透明度

拖动底部工具栏的透明度滑块，范围 20%（较透明）到 100%（完全不透明），实时生效并保存。也可通过右键菜单的透明度选项快速切换。

### 调整字体大小

点击底部工具栏的 **A+** 增大字体，**A-** 减小字体。也可通过右键菜单 → "字体设置" → "增大字体"/"减小字体" 调整。

### 文本格式设置

格式工具栏提供三个按钮：

- **B 按钮** — 加粗/取消加粗选中文本
- **I 按钮** — 斜体/取消斜体选中文本
- **A 按钮** — 打开颜色选择器，设置字体颜色

格式设置以 HTML 富文本形式保存，同时保留纯文本备份用于搜索。

### 隐藏便签

点击底部工具栏的"隐藏"按钮，便签窗口隐藏但保留在系统托盘管理列表中。通过托盘菜单 → 便签列表 → 选择便签 → "打开"即可恢复。

### 窗口吸附与动画

- **窗口吸附**：拖拽便签靠近屏幕边缘（15px阈值）自动吸附对齐；两个便签靠近时边缘自动对齐
- **淡入动画**：便签显示时 200ms 淡入效果
- **淡出动画**：便签隐藏时 150ms 淡出效果

### 设置便签总在最前

勾选底部工具栏的"总在最前"复选框，便签始终显示在其他窗口之上。也可通过右键菜单快速切换。

### 删除便签

- **便签窗口**：点击底部工具栏的"删除"按钮，确认后删除
- **右键菜单**：右键便签 → "删除便签"
- **托盘菜单**：托盘菜单 → 便签列表 → 选择便签 → "删除"

## 高级功能

### 便签搜索

- **打开搜索**：托盘菜单 → "搜索便签" 或按 `Ctrl+Shift+F`
- **搜索方式**：输入关键词，实时搜索便签标题和内容
- **打开便签**：双击搜索结果即可打开对应便签

### 定时提醒

为便签设置定时提醒，到期时系统托盘弹出通知（持续5秒）。

- **设置提醒**：右键便签 → "设置提醒" 或通过托盘菜单
- **提醒类型**：
  - **一次性** — 指定日期和时间，触发后自动清除
  - **每天** — 每天在指定时间提醒
  - **每周** — 每周同一天同一时间提醒
  - **每月** — 每月同一天同一时间提醒
- **防重复**：通过 `last_triggered` 日期跟踪，同一天不会重复提醒
- **提醒按钮状态**：便签上提醒按钮颜色指示提醒状态（无/已设置/已过期）

### 标签管理

便签支持标签分类，便于组织和管理。

- **添加标签**：右键便签 → "设置标签" → 选择已有标签或创建新标签
- **标签颜色**：12种预设颜色可选（红、橙、黄、绿、蓝、紫、青、灰、粉、青蓝、深橙、蓝灰），也可自定义颜色
- **标签管理**：托盘菜单 → "管理标签" → 创建/重命名/删除标签
- **标签芯片**：便签上显示已添加的标签，点击 × 可移除
- **按标签分组**：托盘菜单 → "标签分组" → 选择标签查看该标签下的便签

### 便签模板

快速使用预设模板创建便签。

**5个内置模板**：
| 模板 | 内容 |
|------|------|
| 📋 待办清单 | 复选框格式的任务列表 |
| 📝 会议纪要 | 会议时间、地点、参会人员、议题、决议 |
| 📅 周计划 | 周一至周日的计划安排 |
| 📆 每日日志 | 日期标题 + 事项记录 |
| 💡 头脑风暴 | 自由格式的创意记录区 |

- **从模板创建**：托盘菜单 → "从模板创建..." → 选择模板
- **自定义模板**：在模板对话框中可编辑和保存自定义模板
- **日期占位符**：模板中的 `{date}` 自动替换为当前日期
- **模板存储**：自定义模板保存在 `templates/` 目录

### 导入导出

- **打开**：托盘菜单 → "导入导出"
- **导出格式**：
  - **TXT** — 纯文本导出
  - **Markdown** — 带格式导出
  - **ZIP** — 批量导出所有便签为压缩包
- **导入格式**：支持 TXT 文件导入（单个或批量）
- **后台导出**：ZIP 导出在后台线程执行，不阻塞界面，显示进度条

### 智能格式化

粘贴内容时自动识别格式并美化。

- **支持格式**：JSON、HTML、Markdown、XML、CSS、JavaScript、Python、SQL
- **自动美化**：JSON 自动格式化缩进
- **独立开关**：每个便签可独立开启/关闭自动格式化
- **状态指示**：底部工具栏的格式化开关按钮显示当前状态

### 数据备份

- **打开备份**：托盘菜单 → "备份管理" 或按 `Ctrl+Shift+B`
- **手动备份**：点击"创建备份"，所有便签数据打包为 ZIP 文件
- **自动备份**：程序默认启用自动备份，按配置间隔执行
- **备份恢复**：选择备份文件恢复数据
- **备份清理**：自动保留最近的备份，删除旧文件

## 常见问题

### 1. 如何恢复误删的便签？

便签删除后文件会被移除。建议定期使用备份功能（托盘菜单 → "备份管理"），从备份中恢复数据。

### 2. 主题切换后便签显示异常？

- 确认 `styles/` 文件夹中的 `.css` 主题文件完整无误
- 尝试切换其他主题看是否正常
- 检查主题文件中的样式定义是否完整

### 3. 快捷键不生效？

- 确认没有其他程序占用相同的快捷键组合
- 快捷键在便签窗口获得焦点时均可使用
- 全局快捷键（Ctrl+Shift+N/F/B）在系统任意位置均生效

### 4. 如何添加自定义主题？

1. 在 `styles/` 文件夹中创建新的 `.css` 文件
2. 在文件开头添加主题名称注释：`/* Theme Name: 主题名称 */`
3. 定义便签窗口和控件的样式
4. 重启应用后在设置中找到并选择自定义主题

### 5. 开机自启设置无效？

- 确认您具有修改 Windows 注册表的权限
- 检查注册表 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` 中 `StickyNoteApp` 条目是否存在

### 6. 标签数据存储在哪里？

标签数据存储在应用目录下的 `tags.json` 文件中，便签关联的标签存储在各自 `notes/note_X.json` 的 `tags` 字段中。

### 便签文件存储说明

- **存储路径**：`notes/note_{ID}.json`
- **文件格式**：UTF-8 编码的 JSON

**数据结构示例**：

```json
{
    "title": "工作计划",
    "content": "<p><b>重要</b>：完成项目报告</p><p>参加团队会议</p>",
    "plain_content": "重要：完成项目报告\n参加团队会议",
    "opacity": 0.9,
    "always_on_top": true,
    "geometry": {
        "x": 150,
        "y": 200,
        "width": 400,
        "height": 300
    },
    "theme": "soft_yellow.css",
    "title_font_size": 12,
    "content_font_size": 12,
    "font_settings": {
        "family": "微软雅黑",
        "size": 12,
        "bold": false,
        "italic": false
    },
    "font_color": "#000000",
    "auto_format_enabled": true,
    "tags": ["工作", "重要"],
    "reminder": {
        "enabled": true,
        "datetime": "2026-06-05T09:00",
        "repeat": "daily",
        "message": "检查项目进度",
        "last_triggered": "2026-06-04"
    },
    "template": "todo"
}
```

### 主题文件说明

- **存储路径**：`styles/` 文件夹
- **文件格式**：CSS 样式文件
- **主题识别**：通过 CSS 文件首行的 `/* Theme Name: xxx */` 注释识别主题名称
- **可用主题**：soft_yellow, classic_white, elegant_green, fresh_blue, midnight_black, ocean_teal, sunny_orange, sunset_orange, vibrant_purple, warm_pink
