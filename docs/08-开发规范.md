## 1. 命名规范

### 类命名（PascalCase）
```python
class StickyNote:          # 帕斯卡命名法
class StickyNoteManager:
class SettingsDialog:
class SearchManager:
class WindowPositionManager:
```

### 方法命名（snake_case）
```python
def load_note(self):       # 下划线命名法
def save_note(self):
def update_title(self):
def apply_theme(self):
```

### 变量命名（snake_case）
```python
note_id = 1
notes_dir = './notes'
theme_css = 'soft_yellow.css'
auto_format_enabled = True
```

### 常量命名（UPPER_CASE）
```python
RESIZE_MARGIN = 10        # 全大写下划线
MAX_HISTORY = 50
```

### 私有属性/方法（单下划线前缀）
```python

```

## 2. 注释规范

### 模块文档字符串
```python
# -*- coding: utf-8 -*-
"""
便签搜索功能模块

提供便签搜索和过滤功能
"""
```

### 类文档字符串
```python
class SearchDialog(QDialog):
    """
    便签搜索对话框
    
    提供搜索便签标题和内容的功能，支持实时搜索和结果预览
    """
```

### 方法文档字符串（Google Style）
```python
def perform_search(self, query):
    """
    执行搜索操作
    
    Args:
        query (str): 搜索查询字符串
        
    Returns:
        list: 匹配的便签列表 [(note_id, note_data, is_opened), ...]
    """
```

### 行内注释
```python
# 设置窗口标志为工具窗口，并且不在任务栏显示
flags = Qt.Window | Qt.FramelessWindowHint
```

### 修改标记注释
```python
# **修改部分开始：添加"隐藏"和"删除"按钮**
# ...
# **修改部分结束**
```

## 5. 错误处理规范

### 文件操作
```python
try:
    with open(file_path, 'r', encoding='utf-8') as f:
        data = json.load(f)
except FileNotFoundError:
    data = default_data()
except json.JSONDecodeError:
    QMessageBox.warning(self, '数据错误', 'JSON 格式错误')
except Exception as e:
    QMessageBox.warning(self, '加载错误', f'无法加载文件: {e}')
```

### 用户提示
- 使用 `QMessageBox.information()` 表示成功
- 使用 `QMessageBox.warning()` 表示警告
- 使用 `QMessageBox.question()` 需要确认的操作

### 异常日志
```python
try:
    # 可能出错的操作
except Exception as e:
    print(f"操作失败: {e}")  # 生产环境改用 logging
```

## 8. Git 提交规范

```
<type>: <subject>

[optional body]

类型（type）：
- feat: 新功能
- fix: Bug 修复
- docs: 文档更新
- style: 代码格式（不影响功能）
- refactor: 代码重构
- perf: 性能优化
- test: 测试相关
- chore: 构建/工具变动

示例：
feat: 添加富文本格式工具栏（加粗、斜体、颜色）
fix: 修复 A+/A- 字体大小调整失效
docs: 新增开发文档体系
```

## 9. 性能规范

- 避免在主线程执行耗时操作（如备份使用 `QThread`）
- 使用 `QTimer` 控制自动保存频率
- `UndoRedoManager` 1 秒防抖限制历史记录
- 便签 `save_note()` 全量写入，适用于小数据量场景

---

## 10. 安全规范

- 用户数据仅本地存储，不上传
- 注册表操作需妥善处理权限异常
- 文件路径使用 `os.path.abspath()` 防止路径遍历
- 备份恢复前创建当前备份，防止数据丢失
