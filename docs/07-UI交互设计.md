# 07 - UI 交互设计

## 1. 便签窗口布局

便签采用无边框 + 自定义交互的窗口设计。

```
┌──────────────────────────────────────┐
│  Title 标题 (QLineEdit, 最大50字符)     │  ← 标题栏（可拖拽区域）
├──────────────────────────────────────┤
│                                      │
│  内容编辑区 (QTextEdit, 支持富文本)     │  ← 内容区（可拖拽区域）
│                                      │
│                                      │
├──────────────────────────────────────┤
│ [A-] [A+] | [B] [I] [A]             │  ← 字体工具栏
├──────────────────────────────────────┤
│ 透明度: [═══════╬════]   ☑总在最前   │  ← 控制工具栏
│ ☑智能格式化      [删除] [隐藏]        │
└──────────────────────────────────────┘
```

### 窗口属性
- **窗口标志**：`Qt.Window | Qt.FramelessWindowHint | Qt.Tool`
  - 无边框 + 不在任务栏显示
- **置顶标志**：`Qt.WindowStaysOnTopHint`（可选）
- **最小尺寸**：150 × 150 px
- **默认尺寸**：400 × 300 px
- **调整边缘**：四边 + 四角，10px 感应区域

### 各区域说明

| 区域 | 控件 | 功能 |
|------|------|------|
| 标题栏 | `PlainLineEdit` | 编辑标题，回车不换行，最大 50 字符 |
| 内容区 | `PlainTextEdit` | 多行文本编辑，支持富文本格式 |
| 字体工具栏 | `QPushButton` × 5 | A-/A+ 字体缩放，B 加粗，I 斜体，A 颜色 |
| 控制工具栏 | `QSlider` + `QCheckBox` × 2 | 透明度、置顶、智能格式化 |

## 2. 拖拽与调整大小交互

### 光标样式映射

| 区域 | 光标 | Qt 常量 |
|------|------|---------|
| 左上角 | ↖ | `Qt.SizeFDiagCursor` |
| 右上角 | ↗ | `Qt.SizeBDiagCursor` |
| 左下角 | ↙ | `Qt.SizeBDiagCursor` |
| 右下角 | ↘ | `Qt.SizeFDiagCursor` |
| 左边 | ↔ | `Qt.SizeHorCursor` |
| 右边 | ↔ | `Qt.SizeHorCursor` |
| 上边 | ↕ | `Qt.SizeVerCursor` |
| 下边 | ↕ | `Qt.SizeVerCursor` |
| 内部（非编辑器） | ↖ | `Qt.ArrowCursor`（拖拽） |

### 交互状态机

```
鼠标按下 (mousePressEvent)
  ├─ 点击在编辑器上 → 交给编辑器处理
  └─ 点击在空白区域
      ├─ 边缘 10px 内 → resizing = True, 记录 resize_dir
      └─ 内部区域 → dragging = True, 记录 drag_pos

鼠标移动 (mouseMoveEvent)
  ├─ resizing → perform_resize(globalPos)
  └─ dragging → move(initial_geometry.topLeft + delta)
  └─ 其他 → update_cursor(event) 更新光标样式

鼠标释放 (mouseReleaseEvent)
  → dragging = False, resizing = False
  → 恢复默认光标
  → save_note() 保存新位置
```

## 3. 系统托盘菜单

### 菜单结构

```
┌─────────────────────────────────────┐
│ 添加便签 (Ctrl+Shift+N)              │
│ 搜索便签 (Ctrl+Shift+F)              │
│ 备份管理 (Ctrl+Shift+B)              │
├─────────────────────────────────────┤
│ 便签 ▶                              │
│   ├── 便签1 ▶                       │
│   │   ├── 打开                      │
│   │   └── 删除                      │
│   └── 便签2 ▶                       │
│       ├── 打开                      │
│       └── 删除                      │
├─────────────────────────────────────┤
│ ☑ 开机自启                          │
│ 设置                                │
│ 退出                                │
└─────────────────────────────────────┘
```

### 交互行为
- **双击托盘图标**：打开最近创建的便签
- **隐藏便签**：托盘通知 "便签已隐藏"，持续 2 秒
- **退出应用**：清理快捷键 → 关闭所有便签 → 隐藏托盘 → 退出

## 4. 设置对话框

### 布局

```
┌─────────────────────────────────────────┐
│  设置                              [×]  │
├─────────────────────────────────────────┤
│  [主题设置]  [字体设置]                    │  ← QTabWidget
├─────────────────────────────────────────┤
│                                         │
│  ┌─ 主题选择 ─────────────────────┐     │
│  │ 选择便签默认主题: [下拉选择  ▼] │     │
│  └───────────────────────────────┘     │
│                                         │
│  ┌─ 主题预览 ─────────────────────┐     │
│  │ ┌──────────────────────────┐   │     │
│  │ │  预览标题                 │   │     │
│  │ │                          │   │     │
│  │ │  这是主题预览内容...      │   │     │
│  │ └──────────────────────────┘   │     │
│  └───────────────────────────────┘     │
│                                         │
│                    By：MaWenshui        │
└─────────────────────────────────────────┘
```

字体设置标签页布局类似，包含字体族选择、大小、粗体/斜体开关、预览区域和重置按钮。

### 交互
- **非模态**：`Qt.NonModal`，可同时操作便签窗口
- **主题切换**：下拉选择即实时预览 + 全局应用
- **字体切换**：选择即实时预览 + 全局应用
- **重置字体**：一键恢复 "微软雅黑 12pt"

## 5. 搜索对话框

```
┌─────────────────────────────────────┐
│  搜索便签                           │
├─────────────────────────────────────┤
│ 搜索: [输入关键词搜索...           ] │
│ 找到 3 个匹配的便签                  │
├─────────────────────────────────────┤
│ [已打开] 便签标题1                   │
│    内容预览...                      │
│ [未打开] 便签标题2                   │
│    内容预览...                      │
│ [已打开] 便签标题3                   │
│    内容预览...                      │
├─────────────────────────────────────┤
│ [打开选中]              [关闭]      │
└─────────────────────────────────────┘
```

### 交互
- 输入即实时搜索（最小 1 字符）
- Enter 打开首项，Esc 关闭
- 已打开/未打开状态清晰标注

## 7. 颜色选择对话框

点击字体颜色按钮 `A` 时弹出系统标准 `QColorDialog`：

- 初始颜色：当前字体颜色
- 选择后立即应用到编辑器
- 按钮文字颜色更新为所选颜色
- 选中状态按钮显示蓝色背景

## 8. 快捷键汇总

| 快捷键 | 范围 | 功能 |
|--------|------|------|
| `Ctrl+Shift+N` | 全局 | 创建新便签 |
| `Ctrl+Shift+F` | 全局 | 搜索便签 |
| `Ctrl+Shift+B` | 全局 | 备份管理 |
| `Ctrl+Z` | 编辑器内 | 撤销 |
| `Ctrl+Y` | 编辑器内 | 重做 |
| `Esc` | 搜索对话框 | 关闭对话框 |
| `Enter` | 搜索对话框 | 打开选中便签 |
| 双击 | 托盘图标 | 打开最近便签 |
