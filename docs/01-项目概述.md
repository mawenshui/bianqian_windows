# 01 - 项目概述

## 项目简介

**StickyNote** 是一款基于 PyQt5 开发的 Windows 桌面便签应用程序。它提供轻量级、可定制的便签功能，帮助用户快速记录和管理日常事务、创意笔记和重要信息。

用户可以在桌面上创建多个便签窗口，自由调整大小、位置和透明度，支持富文本编辑，并通过系统托盘进行快捷管理。

## 核心特性

### v1.0.0 - 基础功能（2024年11月）
- 便签创建、编辑、删除
- 无边框窗口 + 自定义拖拽和调整大小
- 窗口透明度调节（20% ~ 100%）
- "总在最前"置顶功能
- 多主题支持（10 套 CSS 主题）
- 系统托盘集成管理
- 开机自启动（Windows 注册表）
- 便签数据 JSON 自动保存

### v1.1.0 - 核心体验优化（2025年6月）
- 便签搜索功能（按标题/内容，支持未打开便签）
- 撤销/重做操作（Ctrl+Z / Ctrl+Y）
- 全局快捷键系统（Ctrl+Shift+N/F/B）
- 智能窗口定位（避免重叠，支持多显示器）
- 数据备份管理（手动/自动备份、恢复）
- features/ 模块化架构
- 字体选择（字体族、大小、粗体、斜体）

### v1.2.0 - 富文本进阶（2025年6月）
- 富文本格式工具栏（加粗、斜体、字体颜色）
- 选择性文本格式应用（仅对选中文本生效）
- 格式按钮状态可视化
- HTML 格式便签内容存储
- 纯文本备份机制（向下兼容 + 搜索索引）
- 智能内容格式化（JSON、HTML、Markdown、Python 等自动识别）
- 字体大小动态调整修复
- 标题栏高度自适应

### v1.3.0 / v1.3.1 - UI/性能优化（2026年6月）
- 右键上下文菜单（复制粘贴/主题/字体/置顶/透明度/标签/提醒）
- 窗口吸附（屏幕边缘 + 便签间边缘对齐，15px阈值）
- 淡入淡出动画（200ms淡入/150ms淡出）
- 异步I/O保存（NoteSaveWorker后台线程 + 500ms防抖）
- 异步I/O加载（NoteLoadWorker后台线程）
- 核心模块拆分（main.py → core/note.py + core/manager.py + core/settings.py）
- Bug修复：复选框状态同步、透明度菜单互斥

### v1.4.0 - 构建与工程化（2026年6月）
- cx_Freeze 替代 PyInstaller 打包方案
- MSI 包更名为 StickyNote 避免中文文件名截断
- 项目目录清理与构建路径规范化
- 新增 tests/ 测试模块（集成测试 + 更新功能测试）
- 新增 tools/ 工具模块（图标生成 + 卸载脚本）

### v1.5.0 / v1.5.1 - 自动更新系统（2026年6月）
- GitHub Release API 自动更新检测与下载
- 后台异步版本检查（QThread），不阻塞主界面
- 检查更新进度条（不确定模式滚动条）
- 取消检查按钮，支持随时中止
- 状态文字提示（连接 → 解析 → 完成）
- 30秒超时适配国内网络环境
- 双链路下载（GitHub主链接 + ghproxy镜像备用）
- 支持 MSI 静默升级和便携版文件替换两种更新方式
- 作者统一为 MaWenshui
- MSI 安装路径优化，自动追加 StickyNote 子目录

## 应用场景

- **日常备忘**：快速记录待办事项、购物清单
- **工作笔记**：会议纪要、项目灵感、代码片段
- **学习笔记**：知识点摘录、复习要点
- **临时记录**：电话号码、地址等临时信息
- **桌面提醒**：重要事项常驻桌面

## 设计理念

1. **轻量简洁** — 最小化资源占用，不打扰用户工作流
2. **即开即用** — 无需复杂配置，启动即可使用
3. **高度可定制** — 主题、字体、透明度、位置完全可控
4. **数据安全** — 本地存储 + 自动备份，数据不离开用户设备
5. **模块化可扩展** — features/ 架构支持轻松添加新功能
