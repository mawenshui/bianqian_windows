# 01 - 项目概述

## 项目简介

**StickyNote** 是一款基于 PyQt5 开发的 Windows 桌面便签应用程序。它提供轻量级、可定制的便签功能，帮助用户快速记录和管理日常事务、创意笔记和重要信息。

用户可以在桌面上创建多个便签窗口，自由调整大小、位置和透明度，支持富文本编辑，并通过系统托盘进行快捷管理。

## 核心特性

- 开机自启动（Windows 注册表）

## 应用场景

- **日常备忘**：快速记录待办事项、购物清单
- **工作笔记**：会议纪要、项目灵感、代码片段
- **学习笔记**：知识点摘录、复习要点
- **临时记录**：电话号码、地址等临时信息
- **桌面提醒**：重要事项常驻桌面

## 设计理念

1. **轻量简洁** — 最小化资源占用，不打扰用户工作流
2. **即开即用** — 无需复杂配置，启动即可使用
3. **高度可定制** — 主题、字体、透明度、位置完全可控
4. **数据安全** — 本地存储 + 自动备份，数据不离开用户设备
5. **模块化可扩展** — features/ 架构支持轻松添加新功能
