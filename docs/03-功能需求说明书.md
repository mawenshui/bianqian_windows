# 03 - 功能需求说明书

## 功能全景

本文档按模块详细描述 StickyNote 的全部功能需求。

---

## 1. 便签管理

### 1.1 创建便签
- **触发方式**：托盘菜单"添加便签"、全局快捷键 `Ctrl+Shift+N`、启动时无便签自动创建
- **行为**：生成唯一 ID，创建新的 `StickyNote` 实例，应用默认主题和字体
- **数据**：生成 `notes/note_{id}.json` 文件

### 1.2 编辑便签
- **标题编辑**：单行输入，最大 50 字符，实时保存
- **内容编辑**：多行富文本编辑，支持加粗/斜体/颜色
- **自动保存**：`textChanged` 信号触发自动保存，无需手动操作

### 1.3 删除便签
- **触发方式**：便签窗口"删除"按钮、托盘菜单右键删除
- **行为**：确认对话框 → 删除 JSON 文件 → 从 Manager 移除 → 关闭窗口
- **保护**：`is_deleted` 标志防止关闭时重复保存

### 1.4 隐藏便签
- **触发方式**：便签窗口"隐藏"按钮、关闭按钮
- **行为**：窗口隐藏（`hide()`），不销毁实例
- **恢复**：托盘菜单 → 便签子菜单 → "打开"

### 1.5 拖拽与调整大小
- **拖拽**：点击便签空白区域（非编辑器）拖动窗口
- **调整大小**：窗口边缘 10px 范围内显示调整光标，支持 8 方向缩放
- **最小尺寸**：150×150 px

### 1.6 透明度调节
- **控件**：水平滑块，范围 20% ~ 100%
- **存储**：opacity 值持久化到 JSON

### 1.7 置顶控制
- **控件**："总在最前"复选框
- **实现**：`Qt.WindowStaysOnTopHint` 标志动态切换

## 2. 富文本格式

### 2.1 加粗 (Bold)
- **按钮**：`B` 按钮，可切换（checked 状态）
- **行为**：
  - 有选中文本 → 切换选中文本的加粗状态
  - 无选中文本 → 切换当前字符格式（影响后续输入）
- **存储**：以 HTML `<b>` 标签保存

### 2.2 斜体 (Italic)
- **按钮**：`I` 按钮，可切换
- **行为**：同加粗，切换 `fontItalic` 属性
- **存储**：以 HTML `<i>` 标签保存

### 2.3 字体颜色
- **按钮**：`A` 按钮，点击弹出 `QColorDialog`
- **行为**：
  - 有选中文本 → 仅对选中文本应用颜色
  - 无选中文本 → 设置当前字符格式
- **按钮样式**：按钮文字显示当前所选颜色的预览
- **存储**：以 HTML `<span style='color: #xxx'>` 保存

### 2.4 字体大小 (A+/A-)
- **按钮**：`A-` 减小、`A+` 增加
- **范围**：最小 6pt
- **联动**：标题栏高度自动适配字体大小

## 3. 搜索功能

### 3.1 搜索入口
- 全局快捷键 `Ctrl+Shift+F`
- 托盘菜单"搜索便签"

### 3.2 搜索范围
- 已打开便签的标题和内容
- 未打开便签的 JSON 文件内容
- 大小写不敏感匹配

### 3.3 搜索结果
- 列表显示，标注 `[已打开]` / `[未打开]` 状态
- 显示内容预览（前 50 字符）
- 双击打开便签，按 Enter 打开首项
- 按 Esc 关闭对话框

---

## 4. 备份管理

### 4.1 自动备份
- 默认每小时自动备份一次
- 备份间隔可配置（1 ~ 24 小时）
- 保留最近 N 个备份（默认 10，可配置 1 ~ 100）
- 自动清理旧备份

### 4.2 手动备份
- 备份管理对话框 → "创建备份"按钮
- 后台线程执行，显示进度条

### 4.3 备份恢复
- 从列表选择备份 → "恢复选中备份"
- 覆盖当前所有数据（有确认提示）
- 建议恢复前先备份当前数据
- 恢复后需重启应用

### 4.4 导入导出
- 导入：选择外部 .zip 文件复制到备份目录
- 导出：将选中备份保存到指定位置

### 4.5 备份内容
- 便签数据（`notes/*.json`）
- 应用设置（`settings.json`）
- 主题文件（`styles/*.css`）

## 5. 全局快捷键

| 快捷键 | 功能 | 类型 |
|--------|------|------|
| `Ctrl+Shift+N` | 创建新便签 | 全局 |
| `Ctrl+Shift+F` | 打开搜索对话框 | 全局 |
| `Ctrl+Shift+B` | 打开备份管理 | 全局 |
| `Ctrl+Z` | 撤销编辑 | 本地 |
| `Ctrl+Y` | 重做编辑 | 本地 |

- 全局快捷键通过 `pywin32.RegisterHotKey` 注册
- 本地快捷键通过 `QKeySequence` 在编辑器 `keyPressEvent` 中处理

## 6. 智能窗口定位

### 6.1 定位策略
- 优先使用历史位置
- 无历史位置时按网格排列（从左到右、从上到下）
- 确保窗口不超出屏幕可用区域
- 避免与已有窗口重叠

### 6.2 位置记忆
- 关闭时保存位置到 `window_positions.json`
- 下次打开优先恢复历史位置
- 验证位置是否仍在屏幕范围内

### 6.3 排列模式
- **网格排列**：`arrange_windows_grid()` — 适合预览所有便签
- **层叠排列**：`arrange_windows_cascade()` — 每个窗口偏移 30px

---

## 7. 撤销/重做

### 7.1 功能
- 标题和内容编辑均支持撤销/重做
- 使用内置 `UndoRedoManager`（undo_redo.py）
- 最多保存 50 个历史状态
- 1 秒防抖，避免频繁保存

### 7.2 注意
当前 `UndoRedoManager` 类已定义但未实际集成到 `StickyNote` 中，便签目前使用 Qt 内置的 undo/redo 支持。

## 8. 智能格式化

### 8.1 自动识别
粘贴内容时自动识别以下格式：
- **JSON**：美化缩进输出
- **HTML**：提取标签结构
- **Markdown**：标记格式类型
- **XML / CSS / JavaScript / Python / SQL**：标记类型

### 8.2 开关控制
- "智能格式化"复选框控制是否启用
- 状态随便签数据持久化
