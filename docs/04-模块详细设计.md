# 04 - 模块详细设计

## 类图总览

```
StickyNoteManager (core/manager.py)
├── notes: dict[int, StickyNote]
├── config: ConfigManager
├── search_manager: SearchManager
├── shortcut_manager: ShortcutManager
├── backup_manager: BackupManager
├── position_manager: WindowPositionManager
├── reminder_manager: ReminderManager
├── tag_manager: TagManager
├── template_manager: TemplateManager
├── link_manager: NoteLinkManager
├── plugin_registry: PluginRegistry
├── plugin_loader: PluginLoader
├── plugin_api: PluginAPI
├── tray_icon: QSystemTrayIcon
└── settings_dialog: SettingsDialog

StickyNote (core/note.py)
├── title_edit: PlainLineEdit (→ UndoRedoLineEdit)
├── text_edit: PlainTextEdit (→ UndoRedoTextEdit)
│   └── formatter: ContentFormatter
├── richtext_actions: RichTextActions
├── markdown_renderer: MarkdownRenderer
├── 字体控制按钮 (A+, A-, B, I, 颜色A, 下划线, 删除线, 对齐, 列表, 高亮)
├── Markdown 模式切换按钮 + 预览面板
├── 标签芯片区 (TagChipWidget)
├── 提醒按钮 (⏰) + 标签按钮 (🏷)
└── 控件 (透明度滑块, 置顶复选框, 格式化复选框)
```

## 1. StickyNote 便签窗口类

**文件**：`core/note.py`

### 属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `note_id` | int | 便签唯一标识 |
| `notes_dir` | str | 便签数据目录绝对路径 |
| `note_file` | str | 便签 JSON 文件路径 |
| `note_data` | dict | 便签数据字典 |
| `theme` | str | 当前主题 CSS 文件名 |
| `is_deleted` | bool | 是否已被删除（防止重复保存） |
| `dragging` | bool | 是否正在拖动 |
| `resizing` | bool | 是否正在调整大小 |
| `resize_dir` | str | 调整大小方向 |
| `font_settings` | dict | 字体设置 |
| `font_color` | str | 字体颜色 hex |
| `title_edit` | PlainLineEdit | 标题编辑器 |
| `text_edit` | PlainTextEdit | 内容编辑器 |

### 关键方法

| 方法 | 说明 |
|------|------|
| `__init__(note_id, notes_dir, manager, theme_css)` | 初始化便签，加载数据，创建 UI |
| `initUI()` | 构建完整界面布局 |
| `load_note() / save_note()` | JSON 数据读写 |
| `apply_theme()` | 读取 CSS 文件并应用样式 |
| `is_dark_theme(css)` | 检测深色主题 |
| `get_adaptive_control_styles(is_dark)` | 获取自适应控件样式 |
| `set_font(font_settings) / apply_font()` | 应用字体设置 |
| `apply_rich_text_format()` | 应用富文本字符格式 |
| `increase_font_size() / decrease_font_size()` | A+/A- 调整字体 |
| `toggle_bold() / toggle_italic()` | 加粗/斜体切换 |
| `choose_font_color()` | 颜色选择 |
| `change_transparency(value)` | 透明度调节 |
| `toggle_always_on_top(state)` | 置顶切换 |
| `toggle_auto_format(state)` | 格式化开关 |
| `delete_note() / hide_note()` | 删除/隐藏便签 |
| `mousePressEvent / mouseMoveEvent / mouseReleaseEvent` | 自定义拖拽和调整大小 |
| `perform_resize(global_pos)` | 执行窗口大小调整 |
| `paintEvent(event)` | 绘制边框 |
| `closeEvent(event)` | 拦截关闭事件（隐藏而非关闭） |

### 属性

| 属性 | 类型 | 说明 |
|------|------|------|

| `search_manager` | SearchManager | 搜索管理器 |
| `shortcut_manager` | ShortcutManager | 快捷键管理器 |
| `backup_manager` | BackupManager | 备份管理器 |
| `position_manager` | WindowPositionManager | 位置管理器 |
| `tray_icon` | QSystemTrayIcon | 系统托盘图标 |
| `tray_menu` | QMenu | 托盘菜单 |
| `notes_menu` | QMenu | 便签子菜单 |
| `settings_dialog` | SettingsDialog | 设置对话框 |
| `config` | ConfigManager | 统一配置管理器 |
| `reminder_manager` | ReminderManager | 提醒管理器 |
| `tag_manager` | TagManager | 标签管理器 |
| `template_manager` | TemplateManager | 模板管理器 |

| `plugin_registry` | PluginRegistry | 插件注册表 |
| `plugin_loader` | PluginLoader | 插件加载器 |

### 关键方法

| 方法 | 说明 |
|------|------|
| `__init__()` | 初始化整个应用 |
| `setup_global_shortcuts()` | 注册全局快捷键 |
| `setup_tray_icon()` | 创建系统托盘和菜单 |
| `load_notes() / load_settings()` | 加载数据和配置 |
| `add_note()` | 创建新便签 |
| `open_note(note_id)` | 打开/显示便签 |
| `delete_note(note_id)` / `remove_note(note_id)` | 删除便签 |
| `generate_note_id()` | 生成不重复的便签 ID |
| `open_settings()` | 打开设置对话框 |
| `set_autostart()` / `check_autostart()` / `toggle_autostart()` | 开机自启 |
| `get_available_themes()` | 扫描 styles/ 获取主题列表 |
| `get_default_theme_css()` / `set_default_theme()` | 默认主题管理 |
| `apply_theme_to_all_notes()` | 全局主题应用 |
| `get_default_font()` / `set_default_font()` | 默认字体管理 |
| `apply_font_to_all_notes()` | 全局字体应用 |
| `update_tray_menu()` | 刷新托盘便签列表 |
| `exit_application()` | 退出应用 |
| `run()` | 启动事件循环 |

## 4. features/ 功能模块

### 4.1 SearchManager & SearchDialog
**文件**：`features/search.py`

| 类 | 职责 |
|----|------|
| `SearchManager` | 管理搜索功能生命周期，提供 `show_search_dialog()` 和 `search_notes()` 接口 |
| `SearchDialog` | 搜索对话框 UI，支持实时搜索、结果列表、双击打开 |

### 4.2 BackupManager & BackupDialog
**文件**：`features/backup.py`

| 类 | 职责 |
|----|------|
| `BackupManager` | 备份核心逻辑：创建/恢复/清理备份、自动备份定时器 |
| `BackupDialog` | 备份管理 UI：设置面板、备份列表、导入导出操作 |
| `BackupWorker` | QThread 子类：后台执行备份操作 |
| `RestoreWorker` | QThread 子类：后台执行恢复操作 |

### 4.3 ShortcutManager
**文件**：`features/shortcuts.py`

| 类 | 职责 |
|----|------|
| `ShortcutManager` | 统一管理全局和本地快捷键 |
| `GlobalShortcutManager` | 通过 pywin32 RegisterHotKey 注册系统级快捷键 |
| `LocalShortcutManager` | 管理应用内快捷键（QKeySequence） |

### 4.4 WindowPositionManager
**文件**：`features/positioning.py`

| 类 | 职责 |
|----|------|
| `WindowPositionManager` | 单例：智能窗口定位、位置记忆、网格/层叠排列、屏幕边界检测 |

### 4.5 UndoRedoManager & 编辑器
**文件**：`features/undo_redo.py`

| 类 | 职责 |
|----|------|
| `UndoRedoState` | 编辑状态快照（文本 + 光标位置 + 时间戳） |
| `UndoRedoManager` | 状态栈管理，支持 undo/redo，最多 50 个状态 |
| `UndoRedoTextEdit` | 扩展 QTextEdit，拦截 Ctrl+Z/Y |
| `UndoRedoLineEdit` | 扩展 QLineEdit，拦截 Ctrl+Z/Y |

### 4.6 ContentFormatter & SmartTextEdit
**文件**：`features/formatter.py`

| 类 | 职责 |
|----|------|
| `ContentFormatter` | 内容类型检测 + 格式化（JSON/HTML/MD/XML/CSS/JS/Py/SQL） |
| `SmartTextEdit` | 扩展 QTextEdit，重写粘贴方法实现自动格式化 |

### 4.7 RichTextActions
**文件**：`features/richtext.py`

| 类 | 职责 |
|----|------|
| `RichTextActions` | 富文本工具栏（下划线、删除线、对齐、列表、高亮、链接、图片） |

### 4.8 MarkdownRenderer
**文件**：`features/markdown_renderer.py`

| 类 | 职责 |
|----|------|
| `MarkdownRenderer` | Markdown → HTML 渲染，支持表格、代码块、目录等扩展语法 |

### 4.9 ImageHandler
**文件**：`features/image_handler.py`

| 类 | 职责 |
|----|------|
| `ImageHandler` | 图片插入策略管理（Base64 嵌入 / 文件引用）、孤立图片清理 |

### 4.10 TagManager & TagChipWidget
**文件**：`features/tag.py`

| 类 | 职责 |
|----|------|
| `TagManager` | 标签 CRUD、颜色管理、便签标签关联、数据持久化 |
| `TagChipWidget` | 标签芯片 UI 控件 |
| `TagEditDialog` | 标签管理对话框 |

### 4.11 ReminderManager & ReminderDialog
**文件**：`features/reminder.py`

| 类 | 职责 |
|----|------|
| `ReminderManager` | 定时提醒管理、周期性检查、托盘通知 |
| `ReminderData` | 提醒数据模型 |
| `ReminderDialog` | 提醒设置对话框 |

### 4.12 TemplateManager & TemplateDialog
**文件**：`features/template.py`

| 类 | 职责 |
|----|------|
| `TemplateManager` | 模板管理、内置模板、自定义模板 |
| `TemplateDialog` | 模板选择/管理对话框 |

### 4.13 NoteLinkManager
**文件**：`features/linking.py`

| 类 | 职责 |
|----|------|
| `NoteLinkManager` | 链接解析、链接索引维护、反向链接查询 |

### 4.14 NoteEncryption
**文件**：`features/encryption.py`

| 类 | 职责 |
|----|------|
| `NoteEncryption` | AES-256-GCM 加密/解密、PBKDF2 密钥派生、Argon2id 密码哈希 |

### 4.15 PDFExporter
**文件**：`features/pdf_exporter.py`

| 类 | 职责 |
|----|------|
| `PDFExporter` | 单便签导出为 PDF 文件（QPrinter） |

### 4.16 ImportExportDialog
**文件**：`features/import_export.py`

| 类 | 职责 |
|----|------|
| `ImportExportDialog` | 导入/导出对话框（TXT/Markdown/ZIP） |
| `ExportWorker` | 导出后台线程 |

### 4.17 UpdateChecker/Downloader
**文件**：`features/updater.py`

| 类 | 职责 |
|----|------|
| `UpdateChecker` | GitHub Release API 版本检查（QThread） |
| `UpdateDownloader` | 更新包下载（主链接 + 镜像备用） |
| `UpdateDialog` | 更新内容展示与确认 |
| `UpdateProgressDialog` | 下载进度对话框 |

### 4.18 ConfigManager
**文件**：`core/config.py`

| 类 | 职责 |
|----|------|
| `ConfigManager` | 单例配置管理器，原子写入、嵌套键路径访问、变更信号通知 |

### 4.19 插件系统
**文件**：`features/plugin_system/`

| 类 | 职责 |
|----|------|
| `PluginBase` | 插件基类接口定义 |
| `PluginRegistry` | 插件注册表管理 |
| `PluginLoader` | 插件目录扫描与加载 |
| `PluginAPI` | 插件与主应用的交互接口 |

### 4.20 云同步
**文件**：`features/sync/`

| 类 | 职责 |
|----|------|
| `SyncEngine` | 同步引擎，协调双向同步 |
| `WebDAVClient` | WebDAV 协议客户端 |
| `LocalSyncClient` | 本地文件夹同步 |
| `ConflictResolver` | 冲突解决策略 |

## 5. 继承关系

```
QLineEdit
  └── UndoRedoLineEdit (features/undo_redo.py)
        └── PlainLineEdit (core/note.py) — 纯文本粘贴

QTextEdit
  └── UndoRedoTextEdit (features/undo_redo.py)
        └── PlainTextEdit (core/note.py) — 纯文本粘贴 + 自动格式化
              └── SmartTextEdit (features/formatter.py) — 智能格式化粘贴

QDialog
  ├── SettingsDialog (core/settings.py)
  ├── SearchDialog (features/search.py)
  ├── BackupDialog (features/backup.py)
  ├── ReminderDialog (features/reminder.py)
  ├── TemplateDialog (features/template.py)
  ├── TagEditDialog (features/tag.py)
  ├── ImportExportDialog (features/import_export.py)
  ├── SyncDialog (features/sync_dialog.py)
  ├── UpdateDialog (features/updater.py)
  └── SetMasterPasswordDialog (features/lock_dialog.py)

QThread
  ├── NoteSaveWorker (core/note.py)
  ├── NoteLoadWorker (core/note.py)
  ├── BackupWorker (features/backup.py)
  ├── RestoreWorker (features/backup.py)
  ├── UpdateChecker (features/updater.py)
  ├── UpdateDownloader (features/updater.py)
  └── ExportWorker (features/import_export.py)
```
