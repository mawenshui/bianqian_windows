# 05 - 数据模型设计

## 1. 便签数据 (note_{id}.json)

存储位置：`notes/note_{id}.json`

### JSON Schema

```json
{
    "title": "便签标题",
    "content": "<!DOCTYPE HTML...>",
    "plain_content": "纯文本备份",
    "opacity": 0.9,
    "always_on_top": true,
    "geometry": {
        "x": 100,
        "y": 100,
        "width": 400,
        "height": 300
    },
    "theme": "soft_yellow.css",
    "title_font_size": 12,
    "content_font_size": 12,
    "font_settings": {
        "family": "微软雅黑",
        "size": 12,
        "bold": false,
        "italic": false
    },
    "font_color": "#000000",
    "auto_format_enabled": true,
    "advanced_toolbar_visible": false,
    "edit_mode": "richtext",
    "markdown_content": "",
    "locked": false,
    "template": "todo"
}
```

### 字段说明

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `title` | string | "便签 {id}" | 便签标题，最大 50 字符 |
| `content` | string | "" | 主要内容，v1.2+ 为 HTML 格式 |
| `plain_content` | string | "" | 纯文本备份，用于搜索和兼容 |
| `opacity` | float | 0.9 | 窗口透明度，范围 0.2 ~ 1.0 |
| `always_on_top` | bool | true | 是否置顶 |
| `geometry.x` | int | 智能定位 | 窗口 X 坐标 |
| `geometry.y` | int | 智能定位 | 窗口 Y 坐标 |
| `geometry.width` | int | 400 | 窗口宽度 |
| `geometry.height` | int | 300 | 窗口高度 |
| `theme` | string | "soft_yellow.css" | 当前主题 CSS 文件名 |
| `title_font_size` | int | 12 | 标题字体大小（旧字段，保留兼容） |
| `content_font_size` | int | 12 | 内容字体大小（旧字段，保留兼容） |
| `font_settings.family` | string | "微软雅黑" | 字体族 |
| `font_settings.size` | int | 12 | 字体大小 (pt) |
| `font_settings.bold` | bool | false | 是否粗体 |
| `font_settings.italic` | bool | false | 是否斜体 |
| `font_color` | string | "#000000" | 字体颜色 Hex |
| `auto_format_enabled` | bool | true | 智能格式化开关 |
| `advanced_toolbar_visible` | bool | false | 高级工具栏是否可见 |
| `edit_mode` | string | "richtext" | 编辑模式（richtext / markdown） |
| `markdown_content` | string | "" | Markdown 原始内容 |
| `locked` | bool | false | 便签是否锁定 |
| `template` | string | — | 创建时使用的模板标识（可选） |

### 存储机制
- 每次编辑触发 `textChanged` → `save_note()`（500ms 防抖）
- `NoteSaveWorker(QThread)` 在后台线程执行 JSON 序列化和文件写入
- 原子写入：先写临时文件再 `os.replace`，防止写入中断损坏

## 2. 应用设置 (settings.json)

存储位置：`settings.json`

```json
{
    "default_theme": "soft_yellow.css",
    "auto_check_update": true,
    "font": {
        "family": "微软雅黑",
        "size": 12,
        "bold": false,
        "italic": false
    },
    "skip_version": "",
    "last_dismissed_version": "",
    "image": {
        "strategy": "base64",
        "max_size_kb": 512
    },
    "security": {
        "master_password_hash": "",
        "master_password_salt": "",
        "require_master_password": false
    },
    "sync": {
        "enabled": false,
        "provider": "webdav",
        "webdav": {
            "url": "",
            "username": "",
            "password_encrypted": "",
            "remote_path": "/stickynote/"
        },
        "local_folder": "",
        "auto_sync": false,
        "sync_interval_minutes": 30
    },
    "plugins": {
        "enabled": true,
        "disabled": [],
        "configs": {}
    }
}
```

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `default_theme` | string | "soft_yellow.css" | 新建便签的默认主题 |
| `auto_check_update` | bool | true | 启动后自动检查更新 |
| `font.family` | string | "微软雅黑" | 默认字体族 |
| `font.size` | int | 12 | 默认字体大小 |
| `font.bold` | bool | false | 默认粗体状态 |
| `font.italic` | bool | false | 默认斜体状态 |
| `skip_version` | string | "" | 跳过的版本号 |
| `last_dismissed_version` | string | "" | 上次“稍后提醒”的版本 |
| `image.strategy` | string | "base64" | 图片插入策略 |
| `image.max_size_kb` | int | 512 | 图片最大大小 (KB) |
| `security.master_password_hash` | string | "" | 主密码哈希 |
| `security.master_password_salt` | string | "" | 主密码盐值 |
| `security.require_master_password` | bool | false | 是否启用主密码 |
| `sync.enabled` | bool | false | 是否启用云同步 |
| `sync.provider` | string | "webdav" | 同步提供商 |
| `sync.webdav.*` | object | — | WebDAV 配置 |
| `sync.auto_sync` | bool | false | 自动同步开关 |
| `sync.sync_interval_minutes` | int | 30 | 同步间隔（分钟） |
| `plugins.enabled` | bool | true | 插件系统开关 |
| `plugins.disabled` | array | [] | 禁用的插件列表 |
| `plugins.configs` | object | {} | 插件配置存储 |

## 3. 窗口位置记录 (window_positions.json)

存储位置：`window_positions.json`

```json
{
    "1": {
        "x": 640,
        "y": 280,
        "width": 640,
        "height": 480
    }
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `{note_id}` | object | key 为便签 ID（字符串） |
| `{note_id}.x` | int | 窗口 X 坐标 |
| `{note_id}.y` | int | 窗口 Y 坐标 |
| `{note_id}.width` | int | 窗口宽度 |
| `{note_id}.height` | int | 窗口高度 |

> ⚠️ **已知问题**：当前文件中存在重复 key "1"，JSON 标准不允许重复 key，会导致加载时仅保留最后一个值。

## 4. 备份配置 (backup_settings.json)

存储位置：`backups/backup_settings.json`

```json
{
    "auto_backup_enabled": true,
    "auto_backup_interval": 3600,
    "max_backup_count": 10
}
```

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `auto_backup_enabled` | bool | true | 是否启用自动备份 |
| `auto_backup_interval` | int | 3600 | 备份间隔（秒） |
| `max_backup_count` | int | 10 | 最大保留备份数 |

## 5. 备份文件 (stickynote_backup_*.zip)

压缩包内部结构：
```
stickynote_backup_20250611_181915.zip
├── notes/
│   ├── note_1.json
│   └── note_2.json
├── settings.json
├── tags.json
└── styles/
    ├── soft_yellow.css
    └── ...
```

命名规则：`stickynote_backup_{YYYYMMDD}_{HHMMSS}.zip`（手动）
`stickynote_backup_imported_{timestamp}.zip`（导入）

## 6. 数据迁移策略

### v1.0 → v1.1 迁移
新增字段：`font_settings`、`auto_format_enabled`
- 设置默认值，不破坏已有数据

### v1.1 → v1.2 迁移
- `content` 字段从纯文本变为 HTML 格式
- 同时保留 `plain_content` 作为备份
- `font_color` 字段新增

### 向下兼容
- `load_note()` 自动处理缺失字段，使用 `default_note_data()` 补齐
- 检测 `content` 是否包含 HTML，选择 `setHtml()` 或 `setText()`

## 8. 链接索引 (links_index.json)

存储位置：`notes/links_index.json`

```json
{
    "1": {
        "outgoing": ["待办清单", "会议纪要"],
        "title": "项目计划"
    },
    "2": {
        "outgoing": [],
        "title": "待办清单"
    }
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `{note_id}` | object | key 为便签 ID（字符串） |
| `{note_id}.outgoing` | array | 该便签链接到的目标标题列表 |
| `{note_id}.title` | string | 该便签的标题 |
