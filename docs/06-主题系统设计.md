# 06 - 主题系统设计

## 概述

StickyNote 采用 **Qt StyleSheet (QSS)** 作为主题系统，基于标准 CSS 语法定义便签和控制的外观。每个主题是一个独立的 `.css` 文件，存放在 `styles/` 目录下。

## 2. CSS 文件规范

### 2.1 主题名称声明

每个 CSS 文件的首行必须包含主题名称注释：

```css
/* Theme Name: 主题显示名称 */
```

程序通过正则 `/* Theme Name: xxx */` 自动提取主题名称，显示在设置界面的下拉列表中。

### 2.2 必须定义的样式选择器

| 选择器 | 说明 |
|--------|------|
| `StickyNote` | 便签主窗口背景色、文字色 |
| `QLineEdit` | 标题输入框样式（背景、边框、字体、颜色） |
| `QTextEdit` | 内容编辑区样式 |
| `QPushButton` | 按钮样式（正常、悬停状态） |
| `QSlider::handle:horizontal` | 透明度滑块手柄 |
| `QCheckBox` | 复选框样式 |
| `QDialog` | 设置对话框背景 |
| `QComboBox` | 下拉框样式 |

### 2.3 示例（柔和黄色主题）

```css
/* Theme Name: 柔和黄色 */

StickyNote {
    background-color: #FFFFCC;
    color: #000000;
}

QLineEdit {
    background-color: #FFFFCC;
    border: 2px solid #FFD700;
    border-radius: 5px;
    font-family: "微软雅黑";
    font-weight: bold;
    text-align: center;
    padding: 5px;
    color: #000000;
}

QTextEdit {
    background-color: #FFFFCC;
    border: 2px solid #FFD700;
    border-radius: 5px;
    padding: 5px;
    font-family: "微软雅黑";
    color: #000000;
}

QPushButton {
    background-color: #FFD700;
    color: #000000;
    border: none;
    border-radius: 5px;
    font-family: "微软雅黑";
    padding: 5px;
}

QPushButton:hover {
    background-color: #FFEA70;
}
```

> **注意**：CSS 文件中不应固定 `font-size`，字体大小由程序代码动态控制。

## 3. 主题加载机制

### 3.1 加载流程

```
StickyNote.apply_theme()
  → 构造 CSS 文件路径：styles/{theme_name}
  → open() 读取 CSS 内容
  → 应用到三个层级：
     1. self.setStyleSheet(style)       — 便签窗口
     2. self.text_edit.setStyleSheet()  — 内容编辑器
     3. self.title_edit.setStyleSheet() — 标题编辑器
  → is_dark_theme() 检测明暗
  → get_adaptive_control_styles() 生成自适应样式
  → apply_adaptive_control_styles() 应用到按钮/分隔符/标签
  → 重新应用字体设置（防止主题覆盖字体）
```

### 3.2 全局主题应用

```
SettingsDialog.on_theme_changed()
  → manager.set_default_theme(css)
  → manager.save_settings()
  → manager.apply_theme_to_all_notes()
      → 每个 StickyNote.set_theme(css)
          → apply_theme()
```

## 6. 自定义主题开发指南

### 创建新主题

1. 在 `styles/` 目录下创建新的 `.css` 文件，如 `my_theme.css`
2. 文件首行添加：`/* Theme Name: 我的主题名称 */`
3. 定义所有必须的样式选择器
4. 启动应用 → 设置 → 主题选择下拉自动出现新主题

### 注意事项
- 使用十六进制颜色代码
- 确保文字与背景有足够对比度
- 不要固定 `font-size`，保持字体灵活性
- 测试所有控件（按钮、滑块、复选框、下拉框）
- 建议同时测试深色和浅色主题下的自适应控件效果
